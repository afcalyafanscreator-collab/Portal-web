# Portal Web by LUTFY

Portal web modern dengan loading screen interaktif.

## 🚀 Fitur

- Loading screen dengan progress bar berwarna ungu
- Navigasi smooth scroll
- Responsive design
- Animasi kartu interaktif
- Konfigurasi terpusat (JSON)
- Struktur file terorganisir

## 📁 Struktur Proyek


## 🛠️ Teknologi

- HTML5
- CSS3 (Animations, Flexbox, Grid)
- JavaScript (ES6+)
- JSON

## 🎨 Desain

- **Warna Loading**: Hitam (background), Ungu (progress bar)
- **Font**: Segoe UI
- **Responsif**: Desktop, Tablet, Mobile

## 📦 Instalasi

1. Clone repository
2. Buka `index.html` di browser
3. Nikmati portal web!
